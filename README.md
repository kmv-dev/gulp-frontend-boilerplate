# gulp-start-frontend
### installation

```
        + download repository
        + npm i
```

### include html

```html
        <!DOCTYPE html>
        <html>
        <head>
            <title>include html</title>
        </head>
        <body>
            <!--=include ./module/header.html -->
            <main class="content">
              content
            </main>
            <!--=include ./module/footer.html -->
        </body>
        </html>
```
